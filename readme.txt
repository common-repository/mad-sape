Плагин для Sape – Mad_Sape. Добавление кода биржи без правки кода темы.

== Description ==

Особенности плагина: 
вывод ссылок Sape в нормальной кодировке сразу после активации плагина и без правки кода.

Инструкция перед установкой: 
скачать архив по ссылке https://www.sape.ru/get_user_files.php со стандартной папкой сапы и закинуть папку в корень сайта, после чего выставить права 777 на эту папку. Например через FileZilla.

== Installation ==

Установка:
 закинуть папку плагина в /wp-content/plugins/
---> активировать плагин
---> перейти в Параметры -> Mad_Sape и ввести имя папки с кодом Sape, которая лежит в корне сайта.

== Frequently Asked Questions ==

= Где выводятся ссылки биржи sape.ru? =

На данный момент вывод ссылок осуществляется только в подвале (footer.php).

= У меня данный плагин вызывает проблемы с вёрсткой. Что мне делать? =

Плагин использует собственный вызов в любой теме сразу после вызова `wp_footer`, поэтому вменяемость отображения блока ссылок на совести разработчиков темы.
